# Kirti Yoga — Product Specification

## Overview

A two-sided web app for a yoga teacher (Kirti) to manage classes, bookings, attendance, and monthly subscriptions for ~30 students. Deployed on Vercel.

## Users

| Role | Description |
|------|-------------|
| **Teacher (admin)** | Kirti — manages everything |
| **Student** | Up to ~30 students — portal for booking, payments, announcements |

---

## Core Features

### 1. Authentication
- Kirti logs in with email + password (single admin account, seeded)
- Students log in with email + password
- Students can self-register via public signup page, OR Kirti adds them manually
- Role-based routing: teacher → `/dashboard`, student → `/student`

### 2. Class Event Management (Teacher)
- Kirti creates a class event: title, date, time, description, capacity (max students)
- Events appear on a schedule visible to all students
- Kirti can cancel or edit events (auto-notifies booked students via WhatsApp)
- Kirti can send a WhatsApp blast to all students about an upcoming class

### 3. Class Booking (Students)
- Students see upcoming classes and book a spot in advance
- Each class has a capacity limit (prevents overcrowding)
- Students can cancel their booking before class starts
- Kirti sees a live headcount per class (booked vs. capacity)
- Booking status: BOOKED | CANCELLED | ATTENDED

### 4. Attendance Tracking
- After class, Kirti marks attendance from the booked-student list
- Students can also self-mark attendance from their portal
- Attendance record: student ↔ event ↔ present/absent

### 5. Payment Model — Monthly Subscription + Registration
- **Registration fee**: one-time fee when a student first joins (configurable amount)
- **Monthly subscription**: fixed monthly fee for 8 classes (2 per week)
- Invoice types: REGISTRATION | MONTHLY
- Invoice is generated by Kirti at the start of each month (or on student registration)
- Student portal shows all pending invoices with a "Pay Now" button

### 6. Payments
- **Online (Razorpay):** Student clicks "Pay Now" → Razorpay checkout → webhook marks invoice paid
- **Offline (UPI walk-in):** Kirti manually marks any invoice as paid from dashboard
- Payment history visible to both teacher and student

### 7. WhatsApp Reminders (very important)
Two types of WhatsApp messages:
- **Class reminders**: Kirti sends upcoming class info to all students (or booked students)
  - Can be triggered per-class or broadcast to all
  - Message: class title, date, time, location/notes
- **Payment reminders**: sent to students with pending invoices
  - Message: amount due, month, payment link
  - Manual trigger per student or bulk per invoice batch
- Uses Twilio WhatsApp API

### 8. Announcements / Updates
- Kirti posts text announcements (title + body)
- Appear on every student's dashboard
- Kirti can edit/delete

### 9. Admin Dashboard (Teacher View)
- Overview: total students, pending invoices, this month revenue
- Class list with headcount (booked/capacity)
- Student list with subscription + payment status
- Quick actions: create class, generate monthly invoices, send WhatsApp blast, add student

### 10. Student Portal
- Upcoming classes with Book / Cancel button
- Pending invoices with Pay button
- Payment history
- Announcements feed
- Own attendance record

## Data Models

### User
- id, name, email, password (hashed), role (TEACHER | STUDENT), phone, createdAt

### ClassEvent
- id, title, description, date, capacity (max seats), cancelledAt, createdAt

### ClassBooking
- id, studentId, classEventId, status (BOOKED | CANCELLED | ATTENDED), createdAt

### Attendance
- id, studentId, classEventId, markedBy (TEACHER | STUDENT), createdAt

### Invoice
- id, studentId, type (REGISTRATION | MONTHLY), month (nullable, for MONTHLY type), amount, status (PENDING | PAID), paidAt, paymentMethod (RAZORPAY | UPI), razorpayOrderId, razorpayPaymentId, createdAt

### Announcement
- id, title, body, createdAt, updatedAt

---

## Success Criteria

- Kirti knows class headcount in advance (no overcrowding, no empty classes)
- Students can book spots and receive WhatsApp reminders about classes
- Kirti generates monthly invoices in one click; students pay online via Razorpay
- Kirti can mark UPI walk-in payments as received
- One-click deploy to Vercel

---

## Out of Scope

- Video/media uploads
- AI chatbot
- Complex analytics / reporting
- Multi-teacher support
- Native mobile app
